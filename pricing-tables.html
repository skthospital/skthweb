Building pricing tables with our predefined classes is quite easy. You'll start with an <ul> and add a class of .pricing-table to it. From there, you just need to add list items inside that represent each piece of the table.

Available HTML class options:
<li class="title">: Creates the styles for a title.
<li class="price">: Add a price that stands out.
<li class="description">: If you need to descript the plan, add this.
<li class="bullet-item">: To call out features, use this list item.
<li class="cta-button">: To add a button inside a list item, use this.
